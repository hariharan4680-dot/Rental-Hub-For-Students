# Rental Hub for Students

A full-stack web application that helps students find and post rental accommodations.

## Features

- Student Registration & Login
- JWT Authentication
- Post Rental Listings
- View All Rentals
- View My Posts
- Update Profile
- MongoDB Atlas Integration

## Tech Stack

- Python
- Flask
- MongoDB Atlas
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/hariharan4680-dot/Rental-Hub-For-Students.git
cd Rental-Hub-For-Students
pip install -r requirements.txt
python server.py
